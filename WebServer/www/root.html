You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /compare?num1=10&num2=10 -- compare two number return true if the equal false otherwise example: http://54.167.126.26:9000/compare?num1=10&num2=10
  • /hash?str1&str2 -- returns the hash code for two given string example: http://54.167.126.26:9000/hash?str1=alhaj&str2=musa
